# Chronicle And Legacy

## 런 종료 유형
- glorious_retirement: 영광스러운 은퇴
- battle_death: 전투 중 사망
- corruption_fall: 오염 몰락
- faction_ascension: 세력 권력 등극
- forbidden_fusion: 금지된 힘과 융합
- vanished_legend: 실종/전설화

## 기록 항목
- 런 요약 문장
- 주요 선택 3개
- 관계 변천 요약
- 완료 퀘스트
- 남긴 태그(명예/악명/오염/축복)

## 다음 세대 이월
- 시작 평판/금화/오염 변형
- 가보 아이템 1개
- 잠금 해제 배경/초상 스타일
- 계보 플래그(부채/흉터/축복/추적)
