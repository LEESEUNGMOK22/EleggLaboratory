# Decision Tiers

## T0 완전 자동
- 반복 전투, 이동, 루팅, 반복 퀘스트
- UI: 로그만 기록

## T1 자동 + 결과 표기
- 작은 관계 변화/보상 선택/소규모 분실/발견
- UI: 결과 카드 없이 로그만

## T2 선택 가능 + 타임아웃 자동
- 중간 기회/리스크, 동료 제안, 명성-보상 트레이드오프
- 조건: 위험도 중간, 장기 플래그 영향 제한
- UI: 팝업 + 타이머 + 프리셋 자동처리

## T3 강제 정지 + 필수 선택
- 세력 확정, 관계 루트 확정, 유물/금지 힘, 계승/은퇴/죽음
- 조건: 장기 플래그, 클래스 방향, 연대기 영향이 큰 경우
- UI: 무조건 일시정지

## 자동 처리 정책
- 이번만: `eventResolutionOverrides[eventId]`
- 이 유형 자동: `eventTypePolicies[eventType]`
- 항상 물어보기: `eventTypePolicies[eventType]=ask`

## 이 문서가 실제 게임 기획/UX에 어떻게 쓰이는지
- 이벤트 분류 기준
- 설정창 자동화 옵션의 데이터 연결 기준
