# Class AI Notes

## 공통
- 클래스는 버튼 입력이 아닌 행동 성향 데이터다.
- 전투/탐험/사회 체크 선호가 함께 정의된다.

## 핵심 메모
- Fighter: 안정 전열, 장비 의존, 리스크 중간
- Rogue: 조건부 폭딜, 은신/기습 선호
- Wizard: 제어/광역 우선, 자원 관리 강함
- Warlock: 지속 압박, 계약 서사와 친화
- Bard: 사회/지원/교란 하이브리드
- Cleric: 회복과 유지력 중심, 신앙 이벤트 가중

## 추천 연결
- `class-ai-presets.json`을 전투 리졸버와 T2 추천 선택지 모두에서 사용

## 이 문서가 실제 게임 기획/UX에 어떻게 쓰이는지
- 클래스별 자동 전투 “느낌” 품질 기준
- 이벤트 추천 문구 생성의 입력 데이터
