# Reward Economy

## 보상 축
- 성장: XP/레벨
- 경제: 골드/소모품/장비
- 서사: 관계/평판/명성/악명
- 위험: 오염/저주/상태이상
- 메타: 연대기 태그/계승 플래그

## 원칙
- 숫자 보상 + 서사 보상 동시 제공
- 위험이 큰 선택은 즉시 보상 + 장기 대가 구조
- 희귀 장비/유물은 수동 확인(T2/T3) 우선

## 계승
- 런 종료 시 `chronicle.entries` 기록
- `inheritancePool`에 다음 세대 시작 보정 저장

## 이 문서가 실제 게임 기획/UX에 어떻게 쓰이는지
- 리워드 테이블 밸런싱 기준
- 연대기 감정선 유지 체크 기준
