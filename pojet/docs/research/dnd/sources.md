마지막 확인: 2026-03-09 (KST)

## 1차 기준 문서 (직접 재사용 가능한 규칙 소스)
- ★ **SRD 5.1 PDF (CC-BY 4.0 공개본)**
  - URL: https://media.wizards.com/2023/downloads/dnd/SRD_CC_v5.1.pdf
  - 성격: 재사용 가능한 규칙 원문(공개 라이선스)
  - 사용 가능 여부: 사용 가능 (원문 대량 복붙은 지양, 구조/규칙/수치 재해석 중심)
  - 주의: 정확한 저작자표시 문구/라이선스 표기는 원문 확인 필요
- ★ **SRD v5.2.1 PDF**
  - URL: https://media.dndbeyond.com/compendium-images/srd/SRD_v5.2.1_CC.pdf
  - 성격: 재사용 가능한 규칙 원문(최신 5.2.x 계열)
  - 사용 가능 여부: 비교 연구/향후 흡수용으로 사용 가능
  - 주의: 본 프로젝트의 기본 구현 베이스는 5.1로 유지
- ★ **SRD Hub / FAQ / Conversion Guide (공식 안내 묶음)**
  - URL: https://www.dndbeyond.com/resources/1781-systems-reference-document-srd
  - 성격: 공식 가이드 허브(규칙 해석, 변환 가이드, 관련 링크)
  - 사용 가능 여부: 규칙 적용 방향/차이점 해석에 사용 가능
  - 주의: 허브 자체 설명 문장을 그대로 제품 문구로 재사용하지 않음

## 정책/브랜드/사용 제한 문서
- ★ **Wizards Fan Content Policy**
  - URL: https://company.wizards.com/en/legal/fancontentpolicy
  - 성격: 팬콘텐츠 정책
  - 사용 가능 여부: 브랜드/표기/배포 정책의 운영 기준으로 사용
  - 주의: 상업적 이용, 로고/트레이드드레스, 공식성 오인 유발 표현 금지 항목 확인 필수
- ★ **Wizards Terms of Use**
  - URL: https://company.wizards.com/en/legal/terms
  - 성격: 서비스 이용약관/상표 고지 포함
  - 사용 가능 여부: 상표/브랜드 문구 사용 시 보수적 기준으로 참고
  - 주의: 법률 문구 해석은 보수적으로 하고, 필요 시 전문 자문 별도

## Creator FAQ 또는 이에 준하는 공식 가이드
- **SRD v5.2.1 출시 공지(공식)**
  - URL: https://www.dndbeyond.com/posts/1839-srd-5-2-1-is-now-available
  - 성격: 공식 배포 공지 + 관련 가이드 링크 안내
  - 사용 가능 여부: 최신 공개 흐름/공식 링크 체인 확인용
  - 주의: 본문은 안내 성격이며 규칙 원문 기준은 PDF
- **D&D Beyond 2024 SRD 커뮤니티 공지(공식 포럼)**
  - URL: https://www.dndbeyond.com/forums/d-d-beyond-general/general-discussion/211919-2024-srd-5-2-1-resources
  - 성격: 공식 커뮤니티 공지, Creator FAQ/변환가이드 링크 소개
  - 사용 가능 여부: 보조 참고
  - 주의: 포럼 게시물 자체를 규칙 원문으로 취급하지 않음
- **Creator FAQ 링크(공식 URL 확인됨, 접근성 변동 가능)**
  - URL: https://www.dndbeyond.com/creator-faq
  - 성격: 공식 Creator FAQ (접근성/표시 상태가 시점별 변동 가능)
  - 사용 가능 여부: 접근 가능 시 정책 확인용
  - 주의: 접근 불가 시 SRD Hub + Fan Policy + Terms를 기준으로 보수 운영

## 학습 참고용이지만 복제 금지인 소스
- D&D Beyond Basic Rules/일반 설명 페이지
  - 이유: 학습에는 유용하지만, 프로젝트의 재사용 가능한 데이터 소스로 간주하지 않음
  - 정책: 설계 감각 참고만 하고 텍스트/표현 복제 금지

## 문서별 사용 가능 여부 요약
- 재사용 가능한 규칙 소스: SRD 5.1 PDF, SRD 5.2.1 PDF
- 운영 정책 소스: Fan Content Policy, Terms, SRD Hub/FAQ
- 학습 참고 전용: Basic Rules/일반 설명 페이지

## 이 문서가 실제 게임 기획/UX에 어떻게 쓰이는지
- 구현팀은 규칙 파싱/시스템 설계 시 SRD 5.1을 우선 참조하고, 5.2.1은 차이점 반영 후보로만 본다.
- UX/카피/세계관 명명 단계에서 Fan Policy/Terms를 체크리스트로 사용한다.
