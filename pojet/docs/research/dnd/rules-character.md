# Rules Character (생성/성장 규칙 정리)

기준: SRD 5.1 우선
마지막 확인: 2026-03-09 (KST)

## 캐릭터 생성 최소 요소
- 종족/오리진 계열
- 클래스
- 능력치 배치
- 배경
- 숙련(기술/도구/언어)
- 시작 장비

## 클래스(프로젝트 요구 12종)
- Barbarian, Bard, Cleric, Druid, Fighter, Monk, Paladin, Ranger, Rogue, Sorcerer, Warlock, Wizard
- 자동 진행 해석 메모:
  - 바바리안: 생존/근접 유지 우선 AI
  - 로그: 조건 충족 시 순간 폭딜(은신/이점 기반)
  - 위저드: 광역/제어/자원 관리형 AI
  - 클레릭: 회복/지원/방어형 우선순위 전환 AI

## 종족/오리진 계열
- 5.1 느낌을 따르되 명칭/설명은 오리지널 세계관으로 재작성
- 자동 진행 해석 메모: 종족은 수동 입력보다 "패시브 보정 + 이벤트 가중치"에 크게 반영

## 배경
- 배경은 기술 숙련 + 초기 소지품 + 서사 씨앗(이상/유대/약점)
- 자동 진행 해석 메모: 배경은 초반 이벤트 풀 가중치와 사회 판정 보정에 직접 연결

## 숙련
- 숙련 여부는 체크의 승률 차이를 만드는 핵심
- 자동 진행 해석 메모: 숙련 데이터는 전투보다 탐험/사회 분기에서 체감되게 설계

## 시작 장비 패키지
- 플레이 템포를 위해 패키지 선택/자동 추천 제공
- 자동 진행 해석 메모: 장비는 "자동 교체 기본 + 잠금 장비 보호" 원칙

## 서브클래스 진입 시점
- 5.1 베이스로 클래스별 전환 시점 감각 유지
- 자동 진행 해석 메모: 서브클래스는 T3 강제 정지 이벤트로 처리

## 이 문서가 실제 게임 기획/UX에 어떻게 쓰이는지
- 캐릭터 생성 화면 설계, 빠른 생성 프리셋, 랜덤 생성 규칙의 기준 문서
- 클래스별 전투 AI 템플릿 정의의 1차 입력 문서
