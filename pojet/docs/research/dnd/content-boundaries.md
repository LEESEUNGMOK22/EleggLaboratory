# Content Boundaries (오리지널화 경계선)

마지막 확인: 2026-03-09 (KST)

## 금지/회피 권장 영역
- 공식 설정명/지명/신명/캠페인 세계의 직접 사용
- 상징적 악역/공식 집단/공식 사건의 재현
- 공식 일러스트/토큰/룰북 문장 대량 인용
- D&D 로고/앰퍼샌드/상표성 높은 브랜딩
- "공식 D&D 게임"처럼 보이게 하는 문구

## 보수적 대체 전략
- 도시/국가/세력: 완전 오리지널 명명 규칙 사용
- 종교/교단/귀족가문: 고유 문법(접두사/접미사)로 일괄 생성
- 몬스터: 일반 판타지 원형으로 재설계 + 우리식 명칭
- 아이템/주문: 기능은 호환 감각, 이름/서사는 독자 IP로 작성
- UI/초상화: 전면 오리지널 아트 디렉션 유지

## 실무 체크리스트
- 명칭 검수: 고유명사 충돌 여부
- 텍스트 검수: 공식 문장 유사도 점검
- 비주얼 검수: 로고/심볼/레이아웃 트레이드드레스 유사성 점검
- 마케팅 검수: 공식성 오인 문구 제거

## 이 문서가 실제 게임 기획/UX에 어떻게 쓰이는지
- 콘텐츠 제작 승인 게이트(문안/아트/UI)의 필수 체크 문서
- 외주 아트/서사 작업 지시서의 금지 항목 기준
