# Event Frequency Targets

## 티어별 목표 범위
- T0: 전체 이벤트의 45%~60%
- T1: 전체 이벤트의 22%~32%
- T2: 런당 3~7회, 평균 8~18틱 간격
- T3: 런당 1~3회, 평균 18~40틱 간격

## 3축 비중 목표
- 전투(combat): 28%~50%
- 탐험(exploration): 20%~42%
- 사회(social): 15%~35%
- 관계(relationship): 전체 이벤트의 8%~30%

## 허용 편차
- 클래스별 생존률 편차: ±15%p 이내 권장
- 프리셋별 생존률 편차: ±18%p 이내 권장
- 특정 배경의 조기 사망률이 전체 평균의 1.8배 초과 시 경고

## 진단 질문
- T2가 너무 자주 떠서 배속 관전 리듬을 끊는가
- T3가 너무 드물어 연대기 기억점이 부족한가
- 관계/사회 이벤트가 전투 로그에 매몰되는가
- 특정 클래스/프리셋에서 이벤트 풀 노출이 급격히 줄어드는가

## 운영 규칙
- 목표 이탈은 즉시 콘텐츠 삭제보다 `발생 확률/쿨다운` 우선 조정.
- T3는 양보다 질: 적게 나오더라도 `chronicle impact`가 반드시 남아야 한다.
- 관계 이벤트는 반복 노출 대신 후속 이벤트 연결률을 올린다.

## 이 문서가 실제 게임 기획/UX에 어떻게 쓰이는지
- 이벤트 스폰 테이블 튜닝의 1차 가드레일.
- 새 지역/세력 추가 시 기존 리듬 붕괴 여부 판단 기준.
